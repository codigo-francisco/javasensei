Este proyecto es un detector de emociones que utiliza
OpenCV y JavaCV para obtener coordenadas de una imagen facial y utiliza un clasificador hecho en la libreria WEKA.

Todo el proyecto es auto contenido, sin embargo se necesitan de las librerias de OpenCV instaladas en el equipo
como lo indica el siguiente tutorial, ya que estas estan realizada en C++, Java CV solo es un wrapper para poder usarlas
en Java.

Basta con seguir el siguiente tutorial para tener OpenCV instalado en windows.
http://docs.opencv.org/doc/tutorials/introduction/windows_install/windows_install.html#windows-installation
