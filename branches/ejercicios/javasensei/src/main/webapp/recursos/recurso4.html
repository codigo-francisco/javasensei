En la etapa de ejecución, el código de bytes es ejecutado por la Máquina Virtual de Java(JVM, por sus siglas en inglés). El código de bytes es el lenguaje de la JVM. Existe una JVM para cada plataforma (para cada sistema operativo); pero, todas las JVM pueden ejecutar el mismo código de bytes. Así, el código de bytes es independiente de la plataforma. Esto hace que los programas Java puedan ser ejecutados en cualquier máquina que disponga de una JVM. Para invocar a la JVM se utiliza el comando java, seguido del archivo que contiene los códigos de bytes pero sin utilizar su extensión. Por ejemplo:

>>java Saludo >>Hola a todos!!
