Identificadores.

Un identificador es el término técnico para el nombre de un componente de programa: el nombre de una clase, el nombre de un método, una variable, etc. Los identificadores deben estar formados enteramente por letras, dígitos y caracteres como signos de moneda ($) y/o guión bajo (_). El primer carácter no debe ser un número. Si un identificador no sigue estas reglas, el programa no compilará.
