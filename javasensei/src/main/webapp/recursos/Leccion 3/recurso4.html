Operadores lógicos.

Las condiciones simples se expresan en términos de los operadores relacionales >, <, >= y <=, y los operadores de igualdad == y !=; cada expresión evalúa sólo una condición. Para evaluar condiciones múltiples en el proceso de tomar una decisión, ejecutamos estas pruebas en instrucciones separadas o en instrucciones if o if/else anidadas. En ocasiones, las instrucciones if requieren condiciones más complejas para determinar el flujo de control de un programa. Java cuenta con los operadores lógicos, AND condicional (&&), el OR condicional (||) y el NOT lógico (!), que nos permite formar condiciones más complejas, al combinar las expresiones simples.
