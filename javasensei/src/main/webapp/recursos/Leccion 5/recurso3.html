Métodos void sin parámetros.

Estos métodos no pueden recibir datos de entrada ni retornar ningún resultado al punto de su invocación. El programa siguiente, TestMetodoVoid.java, ilustra lo mencionado anteriormente.

//Define y utiliza un método void.
public class  TestMetodoVoid {
  public static void  saludo() {
    System. out.println("¡Saludos programador Java!");
  }

  public static void  main(String[] args) {
    saludo();
  } 
}
